# Employee Engagement Survey System

A data engineering capstone project to collect, store, process and visualize employee engagement data.

## Tech Stack
Python, Streamlit, SQLite, Pandas, Plotly

## How to Run
1. pip install -r requirements.txt
2. streamlit run app.py

## Features
- Submit surveys with 6 engagement metrics
- Live analytics dashboard with bar, pie, radar and line charts
- Filter and view all responses
- Export data as CSV
